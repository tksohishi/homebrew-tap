# homebrew-tap

Homebrew tap for [tgcli](https://github.com/tksohishi/tgcli), a CLI tool to read Telegram messages from the terminal.

## Install

```sh
brew install tksohishi/tap/tgcli
```

## Note

The GitHub repo is named `homebrew-tap` because Homebrew requires the `homebrew-` prefix for tap repositories. Users reference it as `tksohishi/tap` (the prefix is stripped automatically).
